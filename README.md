Це фінальний навчальний проєкт з верстання адаптивного веб-сайту, виконаний згідно з макетом з Figma. 

У проєкті використані технології: HTML5, SCSS (змінні, вкладеність, медіа-запити), Bootstrap 5 (grid, navbar, accordion, carousel), Git, GitHub та GitHub Pages.

Сайт адаптовано під такі розміри екранів: мобільна версія від 375px до 500px, планшет від 500px до 1000px, десктоп від 1000px до 1440px. На всіх цих розмірах сайт протестований та коректно відображається.

У хедері реалізовано логотип і навігацію, а також бургер-меню, яке з’являється тільки на екранах менше 500px. Основні секції сайту побудовані з використанням flex, що дозволяє коректно перебудовувати блоки на різних пристроях. 

У блоці Team реалізовано карусель через Bootstrap Carousel: на десктопі відображається 4 елементи горизонтально, на мобільній версії — 4 елементи вертикально. Кількість картинок та контент змінені довільно.

Перед футером додано блок Contact Us, який обгорнутий в Bootstrap Accordion. Усередині розміщено форму та зображення. На мобільній версії елементи відображаються в колонку у зворотному порядку.

SCSS у проєкті організовано через змінні у файлі _variables.scss та розбиття стилів на окремі файли: _header.scss, _content.scss, _footer.scss, які підключені через main.scss. Використані медіа-запити для кожного breakpoint, що дозволило зробити код чистим, структурованим і підтримуваним.
